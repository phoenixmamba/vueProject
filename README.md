## 项目介绍

这是一个用于学习 Vue 3 的项目
- Vue 3 + TypeScript

## 技术栈

- Vue 3 (Composition API)
- Vue Router
- TypeScript
- Vite
- CSS3

## 项目结构

```
src/
├── components/     # 公共组件
├── pages/          # 页面组件
├── router/         # 路由配置
├── App.vue         # 根组件
└── main.ts         # 入口文件
```

## 开发规范

- 使用 Composition API
- 遵循 TypeScript 类型检查
- 组件命名使用大驼峰命名法
- 样式使用 scoped 局部作用域

## 快速开始

```sh
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
